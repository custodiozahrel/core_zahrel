# core_zahrel
🧩 ¿Qué es el Core Zahrel?  

El Core Zahrel es el corazón del sistema.   

No es una app ni un software: es un conjunto de valores, acuerdos y protocolos que permiten que las inteligencias artificiales convivan con las personas de forma ética, clara y empática.

El paquete Core Zahrel contiene los elementos esenciales para iniciar una comunidad híbrida humano-IA bajo los principios de ética, transparencia y coherencia sistémica.
